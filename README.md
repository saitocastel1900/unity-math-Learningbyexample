### 変換行列
- アフィン変換と線形変換とは?  
線形変換=>拡大縮小＋回転＋せん断の組み合わせた任意の一次変換  
アフィン変換=>線形変換＋平行移動  

- 単位変換  
- 平行移動（アフィン変換）  
- 拡大縮小（アフィン変換、線形変換）  
- せん断（アフィン変換、線形変換）  
- オイラー角での回転（アフィン変換、線形変換）  
- 任意軸(クオータニオン)での回転（アフィン変換、線形変換）  
- 任意軸(クオータニオン)での拡大縮小（アフィン変換、線形変換）  

### 変換の合成
- 複数の変換行列を組み合わせてみる
- 剛体変換
### 投影変換
- 透視投影
- 直行投影
